BASIC WEB

russian contructivism

website documentation

The focus of this assingment was to create a mobile view version of a website about a topic connected with history of arts.
The task was also to edit the text about russian constructivism and choose images to make it more clear and easy to follow.
It was the very beggining of getting familiar with coding languagues: HTML and CSS.
We learned how to code using semantic markup so writing the code properly.
Also during this assingment we were introduced to gestalt laws on which the design of a website should be based.

In my design I used different gestalt laws: law of common region, law of similarity and law of proximity.

I started with editting the given text about history of art movement. Then I tried my hand at Adobe XD for the first time where I prepared the prototype of the website.

interest-based website

website documentation

The focus of this assingment was to create both a desktop and mobile view version of a website. Choice of the topic was up to us(hopefully the one that is connected with our interests;)). The first task was to choose one design style as the main theme for the webpage.

I created one of my first moodboards and also I learned how to create a wireframe which is a basic first sketch of how should the content be placed on the page. Another thing I learned was media query. It is adjusting the code in a way that the webpage would be displayed correctly not only on a laptop but also on the phone. Also I learned different multi-device layout patterns.

I decided to use Tiny Tweaks and Column Drop as I wanted to adapt the style almost in the same form as in the desktop view. So the only changes I made were in cutting the background a little bit from both sides and turning the navigation bar into a column. Also I moved the photos closer to the text than before.

I decided to write about Marie Kondo's method on how to declutter space in your house. I've chosen Minimalism as a main design theme for my website as I felt like it matched the topic of my interest the most. Also in this project I started with collecting photos and writing text that would be the content of the page and moved on to creating XD prototype. Drawing wireframes first helped me with organizing the site using coding grid and some flexbox tags. For the first time I decided to organize my stylesheet in only one CSS file which made the coding part easier.
